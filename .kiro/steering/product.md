# Product: Women Safety Assistant

A lightweight, zero-friction personal safety web app built for women. No login, no installation — just open a browser and safety tools are immediately available.

## Core Features

- One-tap emergency actions: open Google Maps, call 112, send WhatsApp SOS
- Keyword-based danger detection from free-text user input
- Full-width panic SOS button that triggers multiple actions simultaneously

## Design Philosophy

- Simplicity is a feature — fewer taps, less thinking under stress
- Dark purple UI: less conspicuous, calming under pressure
- Color-coded actions for instant recognition without reading labels
- Auto-triggers (e.g., open Maps on danger detection) to reduce user burden

## Target Users

Women in potentially unsafe situations who need immediate, reliable access to emergency tools.
